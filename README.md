# Go KVMesh - Lightweight VM Cluster Platform

## Product Positioning
- Debian-based lightweight virtual machine cluster platform
- Self-broadcasting cluster node health monitoring and Unified cluster management from any node
- Automated Cloud-Init official image deployment
- Comprehensive VM monitoring and elastic scaling
- Complete VM backup solutions
- Live migration capabilities
- Software-based solution comparable to Docker
- Multi-interface support: TUI / WebUI / RESTful API
